--- layout: post title: Identical Particles date: '2011-03-02T18:42:00.000-08:00' author: Robert Speare tags: modified_time: '2011-03-02T19:22:53.359-08:00' blogger_id: tag:blogger.com,1999:blog-3410852316732630293.post-1493096198717162118 --- So what if our wavefunction Psi describes, not one, but two particles? For instance, well known closed solutions (however complicated) are known for the hydrogen atom and fairly simple potentials in three dimensions. But say we are interested in tracking the behavior two particles in a system. Now since each wave function represents a statistical distribution, we can simply multiply each particles wave function Psi to get the correct/total probability. this is similar to rolling two die and keeping track of the outcome of both dice rolls, you simply multiply their probability functions. Example

Probability P= P1 * P2
Die Probablity P = P1 * P2 = 1/6 * 1/6
(probability of snake eyes or rolling 12)

Now, there is a problem here unfortunately. Because there is no way for us to make a measurement on ANY quantum mechanical particle without collapsing its wavefunction onto some eigenvector, we can't actually tell which particle is which. We can't simply call 2 electrons, A and B, because we have no way to measure them. So, one must create a total wavefunction that is non-commital as to which particle is which. For instance:

Psi = Psi_1 (electron a) * Psi_2 (electron b)+/- Psi_2 (electron a)*Psi_1(electron b)
this stands for one wavefunction, plus or minus the other. Where the input to each is either the position of electron 'a' or electron 'b'. Now we have constructed a state in which we don't actually have to know which electron is which, we have accounted for both possibilities using the superposition.
