Haven't been introspecting too much recently because of all the studying, but here is what I learned this week that is really cool:

the clock rate, or rate at which the logical processing unit or CPU inside most video game systems nowadays runs so fast that it can literally melt the circuits with such a high frequency. therefore, monster systems like the xbox 360 and playstation 3 use what's called gating to literally shut down portions of the machine when not in use. This saves a ton of power output.

Nowadays, our CPU's run so fast that random access memory or RAM can't keep up because it is physically impossible to transfer information quickly enough from two discrete locations to keep up with the computing power of logical units nowadays. Therefore, register speed or bus speed, the quick acting memory is what seems to be the ultimate obstacle in computing power nowadays. this is called the von neumman bottleneck.

RAM or random access memory is basically the third tier in computer architecture. Computers in the old days had to scroll through all preceding memory in order to arrive at a specific address, making the memory access extremely slow. Random access means that access is completely free, you can simply type in an adress and read the information. Imagine having to scroll through all of your childhood and adolescent memories to arrive at one moment?! that would suck right? The problem with random access memory is that its bitwise memory (0 or 1) is stored and driven by capacitors, which slowly dissipate electricity over time. Therefore, once the power goes out, all ram is lost. This is the difference between sleeping and hibernating for a computer. If am not mistaken, sleeping is when recent information is still in the RAM due to a running current.

When RAM is overloaded, the computer often takes the oldest piece of information and places it on a lower tier of memory. This is called paging, and normally happens when there are a lot of programs running. Repeatedly paging a computer or overloading its RAM is called thrasing, which eventually builds up serious problems.

Now there are some differences between RAM, there is dynamic which requires constant current since it relies on only one transistor, equipped with a capacitor to store bits; and there is static RAM which is much longer lasting but needs 6 transistors for each bit and therefore cannot reach as a high of a density.
