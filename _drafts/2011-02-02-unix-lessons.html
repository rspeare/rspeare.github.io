I hope this helps the common man understand how to get around the bash shell and a Mac's directory system.

Let's start with how the computer is structured. First, everything starts from the root ( / ). This is the ultimate anchor point for all directories on your hard drive. Stored in the root directory are a bunch of different folders/directories that are vital to your operating system. Below the root is the USERS directory, which contains all the user accounts on the computer. For most home PC's and laptops there is only the admin account (the guy with all the access privileges) or the one that you created first when booting up the operating system. There is also a guest account, and the shared account where public files are stored.

Normally when you boot up the bash shell or even MS-DOS, you are started in your User directory. So for example my username is rob, so on start-up I am placed in the following directory: /Users/rob (forward slash separates directory levels)

So, if your on a Mac, start up the terminal application. (The .app file is stored in your Utilities folder, or you can simply type "terminal" into spotlight, which is a great finder tool at the top right of your screen. On start-up you will see a display with your username and then a $ symbol. For example
user$ "enter commands in here"
Now type "pwd" and press enter. This displays your current working directory (or your location in the computers file system).

Wanna move around? First you need to know where to go! To list the working directory contents, or the subfolders type:
user1$ "ls"
The ls command lists files, folder, docs, etc in the current directory. Now, if you want to move into a subfolder named "folder1" type:
user1$ cd folder1
This tells the computer to change directory (cd) to the specified folder. However, we are essentially moving downwards in the file system and will eventually hit a dead end. If you want to move upwards in the directory system you need to type:
user1$ cd ..
The .. means "the folder above" the one your working in now.

Example:
Start terminal

rob$ pwd
/Users/rob
rob$ ls
Documents Pictures
Movies Desktop
Public Music
... (it goes on)
rob$ cd Music
rob$ pwd
/Users/rob/Music
rob$ cd .. ; pwd (this separates two commands into one line, to be executed sequentially left to right)
/Users/rob
rob$ cd .. ; pwd
/Users
rob$ cd .. ; pwd
/

Alright now you know how to get around the file system. Make sure to move around check out all the folders on your computer!
